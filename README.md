# psvrBear #

### What is this repository for? 
make telepresence teddybear using PSVR, threejs, johnny-five, arduino, iRobot create2, peerJS

### How do I get set up? ###

```sh
$ git clone path
$ cd path
$ npm install
$ npm start
```
